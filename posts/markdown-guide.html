Markdown 写作指南

2024年2月1日 写作Markdown工具

什么是 Markdown？

Markdown 是一种轻量级标记语言，由约翰·格鲁伯（John Gruber）在 2004 年创建。它允许人们使用易读易写的纯文本格式编写文档，然后转换成有效的 HTML 文档。

基础语法

标题

使用 # 符号表示标题，数量表示级别：

# 一级标题
## 二级标题
### 三级标题

段落和换行

段落之间用空行分隔。要创建换行，在行尾添加两个空格。

强调

*斜体* 或 _斜体_
**粗体** 或 __粗体__
***粗斜体*** 或 ___粗斜体___

列表

无序列表使用 -、+ 或 *：

- 项目 1
- 项目 2
  - 子项目 2.1
  - 子项目 2.2

有序列表使用数字：

1. 第一步
2. 第二步
3. 第三步

链接和图片

[链接文字](https://example.com)
![图片描述](image.jpg)

代码

行内代码使用反引号：

这是 `code` 示例

代码块使用三个反引号：

```javascript
function hello() {
    console.log("Hello, World!");
}
```

引用

> 这是一段引用文字。
> 可以有多行。

分隔线

---
或
***

进阶技巧

表格

| 表头 1 | 表头 2 | 表头 3 |
|--------|--------|--------|
| 内容 1 | 内容 2 | 内容 3 |
| 内容 4 | 内容 5 | 内容 6 |

任务列表

- [x] 已完成任务
- [ ] 未完成任务
- [ ] 另一个未完成任务

脚注

这是一个有脚注的句子[^1]。

[^1]: 这是脚注的内容。

写作建议

  1. 保持简洁 - Markdown 的精髓就是简洁，不要过度使用格式
  2. 使用标题层级 - 合理使用标题层级，让文档结构清晰
  3. 列表优于长段落 - 信息较多时，使用列表更易读
  4. 代码块标注语言 - 这样能获得语法高亮

常用工具

  • Typora - 所见即所得的 Markdown 编辑器
  • VS Code - 配合 Markdown 插件使用
  • Obsidian - 知识管理工具，支持 Markdown
  • Notion - 支持 Markdown 导入导出

总结

Markdown 是一种简单但功能强大的标记语言，掌握它能让你的写作更加高效。无论是写博客、文档还是笔记，Markdown 都是绝佳的选择。

希望这篇指南对你有帮助！
